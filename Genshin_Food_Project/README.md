# 🍴 提瓦特美食札記

這是一個基於 Node.js 實作的全端 Web 應用程式，旨在建立一個可以自由新增、管理《原神》料理的圖鑑網誌。
## 動機:我喜歡玩原神，想要透過網誌展示原神的料理，可以把喜歡的美食展現出來。

由於本專案包含後端伺服器，請下載後依照以下步驟執行：
1. 開啟終端機執行 `npm install` 安裝必要套件。
2. 執行指令：`node app.js`。
3. 在瀏覽器開啟網址：[http://localhost:3000](http://localhost:3000)

## 核心功能
- **料理新增：** 支援上傳料理圖片、分類（主食/點心/飲品/水果）及詳細介紹。
- **圖片上傳：** 整合 `Multer` 中間件，處理實體圖片儲存於伺服器 `public/uploads` 目錄。
- **資料管理：** 實作 **刪除功能**，可即時移除錯誤資料並同步刪除實體圖片檔案。
- **持久化儲存：** 採用 JSON 檔案系統儲存資料。

## 使用技術
- **前端：** HTML5, CSS3, JavaScript (Vanilla JS), Fetch API
- **後端：** Node.js, Express.js
- **套件：** Multer (圖片處理), Path, FS

## 心得
在開發「提瓦特美食札記」的過程中，我遇到最大的挑戰是 **Node.js 的檔案系統處理**。
1. **EEXIST 錯誤處理：** 最初在自動建立 `uploads` 資料夾時遇到了 `EEXIST` 報錯，這讓我學會使用 `fs.existsSync` 進行檢查。
2. **刪除邏輯：** 實作刪除功能時，除了過濾 JSON 資料，也學習了如何使用 `fs.unlinkSync` 同步清理伺服器上的多餘圖片檔案，維持儲存空間整潔。
