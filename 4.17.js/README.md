# JavaScript 函數與參數練習解答 (習題 6)

本專案包含 10 題 JavaScript 進階語法練習，涵蓋 Callback、IIFE、箭頭函數、陣列處理與非同步操作。

---

## 執行結果一覽

### 1. Callback 基礎實作 (mathTool)
* **邏輯**：傳入匿名函數執行加減運算。
* **輸出**：`15, 5`

### 2. 匿名函數與立即執行 (IIFE)
* **邏輯**：建立私有作用域，外部無法存取 `count`。
* **輸出**：`Count is: 100`

### 3. 箭頭函數與陣列轉換 (map)
* **邏輯**：將價格陣列 `[100, 200, 300, 400]` 打 8 折。
* **輸出**：`[ 80, 160, 240, 320 ]`

### 4. 陣列參數的「破壞性修改」
* **邏輯**：移除最後一項並在開頭加入 "Start"。
* **輸出**：`[ 'Start', 1, 2 ]`

### 5. 函數回傳函數 (Higher-Order Function)
* **邏輯**：建立一個倍數產生器 `multiplier(2)`。
* **輸出**：`20`

### 6. Callback 篩選器 (手寫 filter)
* **邏輯**：手寫迴圈篩選出陣列中大於 7 的數字。
* **輸出**：`[ 8, 12 ]`

### 7. 箭頭函數處理物件 (filter)
* **邏輯**：從使用者陣列中過濾出成年人 (age >= 18)。
* **輸出**：`[ { name: 'Alice', age: 25 } ]`

### 8. 參數傳址陷阱
* **邏輯**：驗證「修改內容」與「重新賦值」的差異。
* **輸出**：
  * `listA: [ 1, 2, 99 ]` (受影響)
  * `listB: [ 3, 4 ]` (不受影響)

### 9. 延遲執行的 Callback (setTimeout)
* **邏輯**：2 秒後將陣列字串組合並印出。
* **輸出**：`Task Completed` (延遲 2 秒出現)

### 10. 綜合應用：計算總價
* **邏輯**：先用 `reduce` 加總，再透過 Callback 扣除 50 元。
* **輸出**：`550`
